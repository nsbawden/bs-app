Guide to Obtaining and Using Your OpenAI API Key

The Bible Study Browser App uses OpenAI's powerful AI models to provide insightful explanations of Bible verses. With your API Key, you can ask the AI to explain selected verses, enhancing your study experience. This guide will walk you through obtaining your key, setting up payment, and adding it to the app.

Obtaining Your OpenAI API Key

  1. Visit OpenAI's platform and sign up or log in.
  2. Once logged in, click on your profile icon in the top-right corner and select "View API keys".
  3. Click "Create new secret key" and give it a name (e.g., "Bible Study App Key").
  4. Copy the key and save it securely. You won't be able to view it again after closing the window.

Setting Up Payment and Understanding Costs

OpenAI requires a payment method for API usage. Here's what you need to know:

  • You must add a payment method to your OpenAI account for the API Key to work.
  • The minimum credit purchase is $5, which is added to your account balance.
  • Costs are based on token usage and vary by the selected model. For example, with gpt-4o-mini (the default model), a typical verse explanation might cost less than $0.01, allowing for hundreds of queries with the minimum $5 credit. Note that selecting a more advanced model like gpt-4o will consume credits faster.
  • You can set usage limits in your OpenAI account to control spending.

For more details on pricing, visit OpenAI's pricing page.

Adding Your API Key to the App

  1. Open the Bible Study Browser App.
  2. Click the gear icon (⚙️) in the top bar to open settings.
  3. Find the "OpenAI API Key" field and paste your key.
  4. Click "Save" to store your key.

Your key will be saved in the app's local storage and used for AI queries.

Additional Information

  • Keep your API Key secure and do not share it with others. The app stores your API Key in your browser's local storage. Ensure your device is secure to protect your key.
  • If you suspect your key has been compromised, you can revoke it from OpenAI's platform and generate a new one.
  • You can manage your API Keys and usage through OpenAI's platform.
  • If you encounter errors with the API Key, ensure it's correctly entered in the app's settings and that your OpenAI account has sufficient credits.

Note: This information is based on OpenAI's policies as of February 2025 and may be subject to change.
